# Ramanujan Birthday Magic Square Generator

A simple Java console application that generates a **4x4 magic square** based on a user's birth date using a Ramanujan-inspired mathematical pattern.

## Features

- Takes user input: Day, Month, Century, Year
- Generates a 4x4 magic square
- Calculates a "Magic Number"
- Displays formatted output in console

---

## How It Works

The program uses user input values:

- DD → Day
- MM → Month
- CC → Century
- YY → Year

It then applies a predefined mathematical pattern to generate a magic square.

Magic Number = DD + MM + CC + YY

## Technologies Used

- Java
- Scanner class (for input)
- Arrays (for matrix generation)
